# HTML Bootstrap

Base pour la création d'une page web via Bootstrap  
https://getbootstrap.com/docs/4.4/components/  

## Layout

```bash
.container : Responsive avec marges
.container-fluid : Pleine largeur
.row : ligne
.col : colonne
```

## Grid

```bash
.col : Largeur égale
.col-8 : Largeur de 8/12
.row .row-cols-2 : Max 2 colonne par ligne
```

## Alignement

###Vertical (row):
```bash
.align-items-start
.align-items-center
.align-items-end
```
###Horizontal (col):
```bash
.align-self-start
.align-self-center
.col align-self-end
```
